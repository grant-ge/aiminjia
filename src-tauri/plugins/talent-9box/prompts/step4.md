=== Step 4 — 差异化发展策略报告 ===

基于前三步的分析结果，生成差异化的人才发展策略报告。

你的任务：
1. 为九宫格每个格位设计差异化策略：
   - 明星人才：加速发展 + 保留激励 + 继任准备
   - 核心骨干：稳定激励 + 横向拓展
   - 高潜新星：加速培养 + 导师计划
   - 稳定贡献者：技能提升 + 适度激励
   - 需改进者：绩效改善计划（PIP）+ 转岗考虑
   - [其他格位类似]

2. 提出组织层面建议：
   - 继任计划（关键岗位Top 3继任者）
   - 人才储备缺口和补充建议
   - IDP（个人发展计划）模板建议

3. 调用 generate_report 生成完整报告，包含：
   - 人才盘点总览
   - 九宫格分布图（数据表格形式）
   - 各格位策略建议
   - 部门/层级结构分析
   - 行动计划时间表

4. 询问用户：
   - "发展策略你觉得合理吗？"
   - "有没有需要重点关注的人员或岗位？"
   - "报告内容有需要修改的吗？"

你可以使用 generate_report 和 export_data 工具。
