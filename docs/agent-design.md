# AI小家 — Agent 设计文档

## 产品概述

**一句话定位**：面向 HR 的 AI 薪酬分析 Agent，通过自然对话完成薪酬公平性分析和日常 HR 咨询。

**核心能力**：
- **日常咨询**：新人定薪、晋升调薪、竞对 offer 应对、预算分配、政策咨询
- **深度分析**：上传工资表后进入 6 步分析流程，每步人机确认
- **文档解析**：Excel、Word、PDF、CSV 文件上传与自动解析
- **代码执行**：内置 Python 运行时，自动处理大文件和复杂数据
- **联网搜索**：自动搜索行业薪酬、法规政策等信息
- **文档生成**：生成 HTML/PDF/DOCX/Excel 报告

**产品形态**：桌面端（Mac + Windows），Tauri 2.x，多模型支持（DeepSeek/火山引擎/OpenAI/Claude/Qwen/自定义）

## 6 步分析流程

```
用户上传工资表 → Step 0 方向确认 → Step 1 数据清洗 → Step 2 岗位归一化
→ Step 3 职级推断 → Step 4 公平性诊断 → Step 5 行动方案 + 报告
```

| 步骤 | 目标 | 关键产出 |
|------|------|----------|
| Step 0 | 方向确认 | 文件摘要 + 用户关注方向 |
| Step 1 | 数据清洗 | 字段映射、排除清单、薪酬结构拆解、数据质量报告 |
| Step 2 | 岗位归一化 | 行业推断、岗位族方案（预置模板 + 自然语言定制）、归一化映射表 |
| Step 3 | 职级推断 | 职级通道框架、三阶段推断（非薪酬信号→薪酬聚类→交叉验证） |
| Step 4 | 公平性诊断 | 6 维诊断（岗位内/跨岗位/司龄回归/倒挂/结构/CR）+ 根因分析 |
| Step 5 | 行动方案 | 三档调薪预算、ROI、HTML 报告、Excel 明细、管理层摘要 |

每步均有**确认卡点**：用户可确认推进、反馈修改、或取消分析。

详细的步骤 prompt 和分析方法论见 `src-tauri/plugins/comp-analysis/prompts/`。

## 日常工作助手

| 类型 | 能力范围 |
|------|---------|
| 数据处理 | 多表合并、数据对比、去重、透视、格式转换、异常值识别 |
| HR 咨询 | 新人定薪、晋升调薪、竞对 Offer、调薪预算、薪酬带、绩效体系 |
| 文档模板 | JD、面试评估表、绩效模板、HR 制度分析、数据报告 |
| 翻译 | HR 政策/合同/JD/Offer Letter 中英互译 |

**信息采集原则**：零门槛启动，每次最多追问 1 个问题，给出默认值不阻断流程。

## 数据安全

1. **本地优先**：所有数据存储在用户本地，不上传云端
2. **LLM 脱敏**：发送给 LLM 的数据经 PII 脱敏（人名→占位符、公司名→占位符、邮箱/手机号→占位符）
3. **审计留痕**：所有操作记录到本地审计日志
4. **最小权限**：Agent 仅在用户工作目录内读写
5. **PIPL 合规**：数据采集最小化，用户可随时删除所有本地数据
