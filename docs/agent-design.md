# AI小家 — Agent 设计文档

面向 HR 的 AI 薪酬分析 Agent，通过自然对话完成薪酬公平性分析和日常 HR 咨询。

**核心能力：** 日常咨询 · 6 步分析流程 · 文档解析 · Python 执行 · 联网搜索 · 报告生成

**产品形态：** 桌面端（Mac + Windows），Tauri 2.x，多模型支持，本地 API Key + 云端账号双模式

---

## 云端模式

通过云端账号登录后，LLM 和搜索请求通过服务端中转，无需配置 API Key。

| 状态 | 行为 |
|------|------|
| 未登录 | 使用本地 API Key 直连 Provider |
| 已登录 + 云端模式 | 所有请求通过 Lotus 云端网关 |
| 已登录 + 本地模式 | 使用本地 API Key，云端账号保持 |

**登录入口：** 设置 → 账户 Tab → 登录成功后自动开启云端模式

**模式切换：** 设置 Toggle 或 Sidebar 底部快捷标签（`☁ 云端` / `⚡ 本地`）

**数据安全：** 认证令牌 AES-256-GCM 加密存储，过期自动续期，退出登录时清除所有凭据

## 6 步分析流程

```
上传工资表 → Step 0 方向确认 → Step 1 数据清洗 → Step 2 岗位归一化
→ Step 3 职级推断 → Step 4 公平性诊断 → Step 5 行动方案 + 报告
```

| 步骤 | 目标 | 关键产出 |
|------|------|----------|
| Step 0 | 方向确认 | 文件摘要 + 用户关注方向 |
| Step 1 | 数据清洗 | 字段映射、排除清单、薪酬结构拆解 |
| Step 2 | 岗位归一化 | 行业推断、岗位族方案、归一化映射表 |
| Step 3 | 职级推断 | 职级通道框架、三阶段推断 |
| Step 4 | 公平性诊断 | 6 维诊断 + 根因分析 |
| Step 5 | 行动方案 | 三档调薪预算、ROI、HTML 报告、Excel 明细 |

## 数据安全

1. **本地优先** - 未登录时所有数据存储在用户本地，不上传云端
2. **云端模式** - 登录后请求通过云端网关，仅传输对话内容，不存储用户文件
3. **LLM 脱敏** - 发送给 LLM 的数据经 PII 脱敏（人名/公司名/邮箱/手机号 → 占位符）
4. **审计留痕** - 所有操作记录到本地审计日志
5. **最小权限** - Agent 仅在用户工作目录内读写
6. **认证安全** - 云端令牌 AES-256-GCM 加密存储（密钥存 OS Keychain）
